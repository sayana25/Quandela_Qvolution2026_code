Classical Benchmarking Strategy

Four baselines across two datasets · Iris (classification) · Swaptions (regression)
Iris
4 features · 3 classes
StandardScaler → sigmoid → (0, π)
80/20 train–test split · stratified
Swaptions
224-dim vol surface · 494 days
PCA → 6 latent scores · StandardScaler
Lag pairs X[t] → y[t+1] · NaN imputation
Iris — Classification Baselines
1
Naive
Logistic Regression
Linear classifier on raw 4-dim features.
Trained with L2 regularisation.
LogisticRegression( max_iter=1000)
Classical linear ceiling. Sets the minimum bar the QRC must clear.
2
Nonlinear
Classical MLP
Two-layer network on raw features.
Architecture: 4 → 64 → 3
Linear(4,64) → ReLU Dropout(0.2) Linear(64,3)
Upper bound for classical models. If QRC ≈ MLP, the reservoir is competitive.
3
Sanity check
Random Projection + Linear
Gaussian random matrix maps raw features to the same dimension as the QRC output. Linear readout trained on top.
W ~ N(0,1), shape (4, feat_dim) X_rand = X @ W Linear(feat_dim, 3)
If QRC ≈ Random Projection, the circuit structure adds nothing beyond noise.
4
Dimensionality
PCA Features + Linear
PCA reduces features to the same dimensionality as QRC output. Linear readout trained on PCA scores.
PCA(n_components=feat_dim) .fit_transform(X) Linear(feat_dim, 3)
Isolates whether QRC adds value beyond classical dimensionality reduction.
Swaptions — Regression Baselines
1
Naive
Persistence Baseline
Predicts tomorrow's vol surface = today's. No model required. Exploits the high autocorrelation of financial time series.
y_pred[t+1] = y[t]
Hardest baseline to beat on near-stationary financial data. Any model must outperform this.
2
Linear
Ridge Regression
Linear regression with L2 regularisation on 6 PCA latent scores. Predicts next-day scores directly.
Ridge(alpha=1.0).fit( scores[:-1], scores[1:])
Classical linear baseline on compressed features. The primary reference point for QRC comparison.
3
Nonlinear
Classical MLP
Three-layer network on 6 PCA scores.
Architecture: 6 → 128 → 64 → 6
Linear(6,128) → ReLU Linear(128,64) → ReLU Linear(64,6)
Tests whether any nonlinear model can improve on Ridge. Sets the classical nonlinear ceiling.
4
Sanity check
Random Projection + Linear
Gaussian random matrix maps 6 PCA scores to the same Fock dimension as the QRC. Linear readout on top.
W ~ N(0,1), shape (6, fock_dim) X_rand = scores @ W Linear(fock_dim, 6)
If QRC ≈ this, the quantum circuit structure contributes nothing over random noise.

Key Diagnostic: Linear vs MLP Readout Gap
For all QRC variants, both a linear readout and an MLP readout are evaluated on the same frozen reservoir features. If |Acc(MLP) − Acc(Linear)| < 0.03 (Iris) or |RMSE(MLP) − RMSE(Linear)| < 0.001 (Swaptions), the reservoir is judged to be doing the nonlinear work and the linear readout is sufficient — which is the theoretically desirable outcome for QRC. A large gap indicates that the reservoir features are not rich enough and the MLP is compensating classically.
Figure 2 — Classical Benchmarking Strategy. Four classical baselines are applied to both datasets. The naive baseline (Logistic Regression / Persistence) establishes the minimum the QRC must beat. The linear baseline (Logistic Regression / Ridge) tests whether nonlinearity is needed at all. The nonlinear baseline (Classical MLP) provides the classical performance ceiling. The random projection baseline is the critical sanity check: if QRC performance matches a random Gaussian projection of the same output dimension, the quantum circuit structure contributes nothing. All baselines are evaluated using the same metrics as the QRC models — accuracy for Iris and RMSE / R² for Swaptions — enabling direct comparison.
